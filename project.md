## Hoe werkt het? (flow)

1. Gebruker selecteert welke gebruiker hij is (select box)
1.1. Gebruiker komt toe op dashboard waar hij kan kiezen op wat hij wil stemmen (Algemene thema's, Ronde thema's, Tafelrondes, Schiftingsvragen)
2. Gebruker kiest wat hij wil stemmen
3. Gebruker stemt (Voor de 3 dingen die hij kan stemmen)
4. Stemmen worden opgeslagen op excel (script.google.com)
